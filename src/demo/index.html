🎨 maUI 组件库演示

基于原生 Web Components 构建的现代化 UI 组件库，使用 TypeScript + SCSS + Webpack 开发。

按钮组件 (ma-button)

基础用法

默认按钮 主要按钮 次要按钮 危险按钮 幽灵按钮
<ma-button>默认按钮</ma-button> <ma-button variant="primary">主要按钮</ma-button> <ma-button variant="secondary">次要按钮</ma-button> <ma-button variant="danger">危险按钮</ma-button> <ma-button variant="ghost">幽灵按钮</ma-button>

不同尺寸

小按钮 中按钮 大按钮
<ma-button size="small" variant="primary">小按钮</ma-button> <ma-button size="medium" variant="primary">中按钮</ma-button> <ma-button size="large" variant="primary">大按钮</ma-button>

禁用状态

禁用按钮 禁用主要按钮 禁用危险按钮
<ma-button disabled>禁用按钮</ma-button> <ma-button variant="primary" disabled>禁用主要按钮</ma-button>

加载状态

加载中... 点击测试加载
<ma-button loading variant="primary">加载中...</ma-button>

事件测试

点击我
等待点击事件...
// 监听自定义事件 document.querySelector('#eventBtn').addEventListener('ma-click', (event) => { console.log('按钮被点击了！', event.detail); });

输入框组件 (ma-input)

基础用法

<ma-input label="用户名" placeholder="请输入用户名"></ma-input> <ma-input label="邮箱" type="email" placeholder="请输入邮箱"></ma-input> <ma-input label="密码" type="password" placeholder="请输入密码"></ma-input> <ma-input label="电话" type="tel" placeholder="请输入电话号码"></ma-input>

不同尺寸

<ma-input label="小尺寸" size="small" placeholder="小尺寸输入框"></ma-input> <ma-input label="中等尺寸" size="medium" placeholder="中等尺寸输入框"></ma-input> <ma-input label="大尺寸" size="large" placeholder="大尺寸输入框"></ma-input>

样式变体

<ma-input label="外边框样式" variant="outlined" placeholder="外边框样式"></ma-input> <ma-input label="填充样式" variant="filled" placeholder="填充样式"></ma-input> <ma-input label="标准样式" variant="standard" placeholder="标准样式"></ma-input>

状态演示

<ma-input label="禁用状态" disabled placeholder="禁用的输入框"></ma-input> <ma-input label="只读状态" readonly value="只读文本内容"></ma-input> <ma-input label="必填字段" required placeholder="必填字段，不能为空"></ma-input> <ma-input label="错误状态" error="输入内容有误，请重新输入" value="错误的内容"></ma-input>

功能特性

<ma-input label="可清空" clearable placeholder="可清空的输入框" value="可以清空我"></ma-input> <ma-input label="密码可见" type="password" placeholder="密码可见性切换"></ma-input> <ma-input label="辅助文本" helper-text="这是一个提示性的辅助文本"></ma-input> <ma-input label="字符限制" maxlength="10" helper-text="最多只能输入10个字符"></ma-input>

验证功能

<ma-input label="邮箱验证" type="email" required pattern="[a-z0-9._%+-]+@[a-z0-9.-]+\.[a-z]{2,}$"></ma-input> <ma-input label="数字输入" type="number" min="0" max="100"></ma-input> <ma-input label="长度验证" minlength="6" maxlength="20"></ma-input>

事件测试

等待输入事件...
// 监听输入事件 document.querySelector('#eventInput').addEventListener('ma-input', (event) => { console.log('输入内容改变:', event.detail.value); }); document.querySelector('#eventInput').addEventListener('ma-change', (event) => { console.log('输入完成:', event.detail.value); }); document.querySelector('#eventInput').addEventListener('ma-valid', (event) => { console.log('验证通过:', event.detail.result); });
