🚀 快速开始

安装

# 使用 npm
npm install maui

# 使用 yarn
yarn add maui

基础使用

maUI 基于原生 Web Components 构建，无需任何框架依赖。只需引入组件即可使用：

// 引入组件
import 'maui/ma-button';
import 'maui/ma-layout';

// 在 HTML 中使用
<ma-button variant="primary">点击我</ma-button>

<ma-layout has-sider="true">
  <div slot="header">Header</div>
  <div slot="sider">Sidebar</div>
  <div slot="content">Content</div>
</ma-layout>

特性

  • 🎯 原生 Web Components - 无框架依赖，与任何前端框架兼容
  • 🎨 现代化设计 - 基于现代设计系统，支持主题定制
  • 🔧 TypeScript 支持 - 完整的类型定义，更好的开发体验
  • 📱 响应式设计 - 移动端友好，自适应各种屏幕尺寸
  • ♿ 可访问性 - 遵循 ARIA 标准，支持键盘导航和屏幕阅读器
  • 🚀 高性能 - Shadow DOM 隔离，优化的渲染性能

浏览器支持

maUI 支持所有现代浏览器：

  • Chrome 63+
  • Firefox 63+
  • Safari 13+
  • Edge 79+
