🎨 主题定制

CSS 自定义属性

maUI 使用 CSS 自定义属性（CSS Variables）来实现主题定制。你可以通过覆盖这些变量来自定义组件的外观：

:root {
  /* 主色调 */
  --ma-primary-color: #007bff;
  --ma-secondary-color: #6c757d;
  --ma-success-color: #28a745;
  --ma-danger-color: #dc3545;
  --ma-warning-color: #ffc107;

  /* 字体 */
  --ma-font-family: 'Inter', -apple-system, sans-serif;
  --ma-font-size-base: 14px;

  /* 间距 */
  --ma-spacing-xs: 4px;
  --ma-spacing-sm: 8px;
  --ma-spacing-md: 16px;
  --ma-spacing-lg: 24px;

  /* 圆角 */
  --ma-border-radius: 6px;

  /* 阴影 */
  --ma-box-shadow: 0 2px 8px rgba(0, 0, 0, 0.1);
}

深色主题

maUI 内置了对深色主题的支持。添加 data-theme="dark" 到 body 或容器元素：

<body data-theme="dark">
  <ma-button variant="primary">深色主题按钮</ma-button>
</body>

自定义主题示例

创建你自己的主题：

/* 紫色主题 */
.purple-theme {
  --ma-primary-color: #6f42c1;
  --ma-primary-hover: #5a359a;
  --ma-primary-active: #4e2d83;

  --ma-secondary-color: #e83e8c;
  --ma-secondary-hover: #d61a7b;
  --ma-secondary-active: #c01568;
}

/* 大字体主题 */
.large-font-theme {
  --ma-font-size-base: 16px;
  --ma-font-size-sm: 14px;
  --ma-font-size-lg: 20px;
}

/* 圆角主题 */
.rounded-theme {
  --ma-border-radius: 20px;
  --ma-border-radius-sm: 12px;
  --ma-border-radius-lg: 28px;
}
