Button 按钮

常用的操作按钮，支持多种尺寸、状态和样式。基于 Web Components 实现，具有良好的封装性和复用性。

基本用法

默认按钮 主要按钮 次要按钮 危险按钮 幽灵按钮
HTML 
<ma-button>默认按钮</ma-button>
<ma-button variant="primary">主要按钮</ma-button>
<ma-button variant="secondary">次要按钮</ma-button>
<ma-button variant="danger">危险按钮</ma-button>
<ma-button variant="ghost">幽灵按钮</ma-button>

按钮尺寸

小型按钮 中等按钮 大型按钮
HTML 
<ma-button size="small" variant="primary">小型按钮</ma-button>
<ma-button size="medium" variant="primary">中等按钮</ma-button>
<ma-button size="large" variant="primary">大型按钮</ma-button>

禁用状态

禁用按钮 禁用主要按钮 禁用危险按钮
HTML 
<ma-button disabled>禁用按钮</ma-button>
<ma-button variant="primary" disabled>禁用主要按钮</ma-button>
<ma-button variant="danger" disabled>禁用危险按钮</ma-button>

加载状态

加载中 提交中 点击测试加载
HTML 
<ma-button loading>加载中</ma-button>
<ma-button variant="primary" loading>提交中</ma-button>

<!-- JavaScript 控制加载状态 -->
<script>
const btn = document.querySelector('ma-button');
btn.loading = true; // 开启加载
btn.loading = false; // 关闭加载
</script>

事件处理

点击测试事件
事件日志：
点击按钮查看事件详情
JavaScript 
// 监听自定义事件
button.addEventListener('ma-click', (event) => {
  console.log('按钮被点击', event.detail);
});

button.addEventListener('ma-focus', (event) => {
  console.log('按钮获得焦点', event.detail);
});

button.addEventListener('ma-blur', (event) => {
  console.log('按钮失去焦点', event.detail);
});

API 文档

属性 (Attributes)

属性名 类型 默认值 说明
size small | medium | large medium 按钮尺寸
variant primary | secondary | danger | ghost primary 按钮类型
type button | submit | reset button 按钮的 HTML type 属性
disabled boolean false 是否禁用
loading boolean false 是否加载中

事件 (Events)

事件名 说明 回调参数
ma-click 点击按钮时触发 event.detail.originalEvent
ma-focus 按钮获得焦点时触发 event.detail.originalEvent
ma-blur 按钮失去焦点时触发 event.detail.originalEvent

方法 (Methods)

方法名 说明 参数
focus() 使按钮获得焦点 —
blur() 使按钮失去焦点 —
click() 模拟点击按钮 —
